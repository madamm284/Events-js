# JavaScript DOM Events Playground

A small practice project exploring **JavaScript DOM manipulation and event handling**.

This mini playground demonstrates how JavaScript can dynamically interact with HTML elements in response to different user events.

## Features

* **Click Event**

  * Button changes the text of a heading.
  * Additional **Undo button** resets the text to its original state.

* **Live Input Display**

  * Text typed into an input field appears instantly on the page.
  * Includes a dynamically generated **Reset button**.

* **Mouse Events**

  * A box changes color when the mouse **enters** and **leaves** the element.

* **Keyboard Event**

  * Pressing **Enter** inside an input field triggers an alert.

## Concepts Practiced

* `addEventListener`
* DOM selection (`getElementById`, `querySelector`)
* Creating elements with `createElement`
* Dynamically adding elements using `appendChild`
* Handling keyboard, mouse, and input events
* Basic state resetting
